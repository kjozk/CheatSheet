# Markdownのチートシート

## 目的

コピペしてそのまま使える形でまとめる。

## ブロック要素

### 見出し / Header

文頭に`#`を書くと見出しになります。
HTMLの`<h1>`～`<h6>`タグ相当に相当し、表示環境によってはアウトライン表示されます。

**記述例**

```
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
```

また以下のようにイコールを書けば`<h1>`、ハイフンを書けば`<h2>`になります。

```
見出し1
=======

見出し2
-------
```

見づらくなるので表示例は割愛します。

### 引用 / Blockquote

行頭に `>` を記載すると引用になります。
HTMLの`<blockquote>`タグに相当します。

**記述例**

```
>> 引用の引用
>
> 引用
```

**表示例**

>> 引用の引用
>
> 引用


### コード / Code

HTMLの`<code>`タグに相当します。

#### インラインコード

文中にコードを埋め込むインラインコードの場合は、バッククオートで囲みます。

**記述例**
```
ここに`コード`を書きます。
```

**表示例**
ここに`コード`を書きます。


#### 言語指定

コードが複数行にまたがる場合は、バッククオート3つ```で囲みます。
また、バッククオートの行に言語名を明記することもできます。

**表示例**

```C++
#include <memory>

class Bird {
public:
    virtual void Fly() = 0;
};

class Crow : public Bird {
public:
    virtual void Fly();
};
```

```Python
import io

class Bird:
    def Fly(self):
        pass

class Penguin(Bird):
    def Fly(self):
        pass
```

### 改行 / Break

HTMLの`<br>`タグを使います。

### 水平線 / Horizontal Rule

アスタリスク3つ以上`***`や`* * *`、ハイフン3つ以上`---`や`- - -`、アンダースコア3つ以上`___`や`_ _ _`を記載すると水平線になります。
HTMLの`<hr>`タグに相当します。
ただしハイフン`---`は`<h2>`が優先されるので、使い方に注意が必要です。

**記述例**

```
アスタリスク3つ
***
* * *
ハイフン3つ
- - -
アンスコ3つ
___
_ _ _
```
**表示例**

アスタリスク3つ
***
* * *
ハイフン3つ
- - -
アンスコ3つ
___
_ _ _

## リスト

### 順序なしリスト / Unordered List

行頭にアスタリスク`*`、ハイフン`-`、プラス`+`を書くと順序なしリストになります。
HTMLの`<ul>`タグに相当します。
インデントするとネストされる。

**記述例**

```
* 順序なしリスト1
    * 順序なしリスト1.1
    * 順序なしリスト1.2
        * 順序なしリスト1.2.1
    * 順序なしリスト1.3
* 順序なしリスト2
    * 順序なしリスト2.1
* 順序なしリスト3
* 順序なしリスト4
```

**表示例**

* 順序なしリスト1
    * 順序なしリスト1.1
    * 順序なしリスト1.2
        * 順序なしリスト1.2.1
    * 順序なしリスト1.3
* 順序なしリスト2
    * 順序なしリスト2.1
* 順序なしリスト3
* 順序なしリスト4

### 順序ありリスト / Ordered List

行頭に`1.`を書くと順序ありリストになります。
HTMLの`<ol>`タグに相当します。
すべて`1.`でも勝手にインクリメントされます。

```
1. 順序ありリスト1
    1. 順序ありリスト1.1
    1. 順序ありリスト1.2
        1. 順序ありリスト1.2.1
    1. 順序ありリスト1.3
1. 順序ありリスト2
    1. 順序ありリスト2.1
1. 順序ありリスト3
1. 順序ありリスト4
```

1. 順序ありリスト1
    1. 順序ありリスト1.1
    1. 順序ありリスト1.2
        1. 順序ありリスト1.2.1
    1. 順序ありリスト1.3
1. 順序ありリスト2
    1. 順序ありリスト2.1
1. 順序ありリスト3
1. 順序ありリスト4


## 書式

### 強調 / Emphasis & Strong

アスタリスク1つ`*`やアンダースコア1つ`_`で囲むと強調になります。
アスタリスク2つ`**`やアンダースコア2つ`__`で囲むと強い強調になります。

強調とはHTMLの`<em>`タグのことで、一般的には_イタリック体_で表示されます。
強い強調とは`<strong>`タグのことで、一般的には_ボールド体_で表示されます。

**記述例**

```
アスタリスクやアンダースコアで囲むと*強調*になります。
アスタリスクやアンダースコア2つで囲むと**強い強調**になります。
```
**表示例**

アスタリスクやアンダースコアで囲むと*強調*になります。
アスタリスクやアンダースコア2つで囲むと**強い強調**になります。

### 抹消線 / Strike

HTMLタグ`<s>`で囲むと抹消線が表示されます。

**記述例**

`<s>抹消線</s>`

**表示例**

<s>抹消線</s>

### 下線 / Underline

HTMLタグ`<u>`で囲むと下線が表示されます。

**記述例**

`<u>下線</u>`

**表示例**

<u>下線</u>

※GitHubでは非対応

### 上付き・下付き / Superscript, Subscript

`上付き<sup>[1]</sup>下付き<sub>[2]</sub>`

上付き<sup>[1]</sup>下付き<sub>[2]</sub>


### 文字色等

環境によっては`<font>`タグや`<span style="">`といったスタイルシートが使える場合もあります。


## ハイパーリンク / Hyperlink

### 基本的な書き方

**記述例**

```
[READMEへ](README.md)
```

**表示例**

[READMEへ](README.md)

### タイトルの書き方

HTMLタグ`<a>`タグのtitle属性に相当します。

**記述例**

```
[READMEへ](README.md "タイトル")
```

**表示例**

[READMEへ](README.md "タイトル")


### 脚注の書き方

```
[README][linkref]
[りーどみー][linkref]
[読んで！][linkref]

すべて同じリンクになります。

[linkref]: README.md "タイトル"
```

[README1][linkref]
[りーどみー][linkref]
[読んで！][linkref]

すべて同じリンクになります。

[linkref]: README.md "タイトル"
