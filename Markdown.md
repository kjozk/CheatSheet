# Markdownのチートシート


## ブロック要素

### 見出し

HTMLの`<h1>`～`<h6>`タグ相当に相当する。
アウトラインとして認識される。

**記述例**
```
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
##### 見出し5
###### 見出し6
```
表示例は割愛


### 引用

行頭に `>` を記載すると引用になる。

**記述例**
```
>> 引用の引用
>
> 引用
```
**表示例**
>> 引用の引用
>
> 引用


### コード

### 水平線
アスタリスク3つ`***`、ハイフン3つ`---`、アンダースコア3つ`___` を記載すると水平線になる。
ただしハイフン3つ`---`の上は空行にしないと見出しになる。

**記述例**
```
アスタリスク3つ
***
ハイフン3つ

---
アンスコ3つ
___
```
**表示例**
アスタリスク3つ
***
ハイフン3つ

---
アンスコ3つ
___



## リスト

### 箇条書き

行頭にアスタリスク`*`、ハイフン`-`、プラス`+`を書くと箇条書きになる。
インデントするとネストされる。

**記述例**
```
* 箇条書き1
    * 箇条書き1.1
    * 箇条書き1.2
        * 箇条書き1.2.1
    * 箇条書き1.3
* 箇条書き2
    * 箇条書き2.1
* 箇条書き3
* 箇条書き4
```

**表示例**
* 箇条書き1
    * 箇条書き1.1
    * 箇条書き1.2
        * 箇条書き1.2.1
    * 箇条書き1.3
* 箇条書き2
    * 箇条書き2.1
* 箇条書き3
* 箇条書き4

### 番号付き

行頭に`1.`を書くと番号付きになる。
勝手にインクリメントされる。
インデントするとネストされる。

```
1. 番号付き1
    1. 番号付き1.1
    1. 番号付き1.2
        1. 番号付き1.2.1
    1. 番号付き1.3
1. 番号付き2
    1. 番号付き2.1
1. 番号付き3
1. 番号付き4
```

1. 番号付き1
    1. 番号付き1.1
    1. 番号付き1.2
        1. 番号付き1.2.1
    1. 番号付き1.3
1. 番号付き2
    1. 番号付き2.1
1. 番号付き3
1. 番号付き4


## 書式

### イタリック体
アスタリスク1つ`*`で囲むとイタリックになります

**記述例**
```
アスタリスクで囲むと*italic*になります
```

**表示例**
アスタリスクで囲むと*italic*になります

### 太字
アスタリスク2つ`**`で囲むと太字になります

**記述例**
```
アスタリスクで囲むと**太字**になります
```

**表示例**
アスタリスクで囲むと**太字**になります




### 抹消線（Strike）
**記述例**
`<s>下線</s>`

**表示例**
<s>下線</s>

※GitHubでは非対応

### 下線（Underline）
**記述例**
`<u>下線</u>`

**表示例**
<u>下線</u>

※GitHubでは非対応

### 上付き・下付き
`上付き<sup>[1]</sup>下付き<sub>[2]</sub>`

上付き<sup>[1]</sup>下付き<sub>[2]</sub>
